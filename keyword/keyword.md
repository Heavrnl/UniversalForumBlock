# 交易&抽奖

## 正则表达式
```
(?=.*合租)(?=.*(?:入口|落地))
.*\d+\s*出.*
.*\d+\s*包.*
.*\d+\s*收.*
.*\d+\s*抽.*
^出.*
^收.*
^抽.*
[()\[\]【】（）「」][^()\[\]【】（）「」]*?(收|出|抽|送)[^()\[\]【】（）「」]*?[()\[\]【】（）「」]
(?=.*(?:域名|米|已|甩|降价|不|试|贱|有没有|特价|同|打包))(?=.*(?:出|卖|收|抽))
.*(?:收|出|卖|买|抽)(?:一个|几个|\d*u|掉|.*机|.*鸡).*
.*(?:原)(?:邮|油).*
.*(?:低|溢|折|原|带|降)(?:价).*
.*(?:明|命)(?:盘).*
.*(?:抽|开)(?:奖).*
^\d.{0,4}\d.*?(?:出|收).*$
```
